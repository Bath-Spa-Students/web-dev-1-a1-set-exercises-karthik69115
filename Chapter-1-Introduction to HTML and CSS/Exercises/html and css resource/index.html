HTML Tags

The below mentioned are some of the important html tags:

  • <B>: used to make the text BOLD. 
    <B>Welcome to BathSpa.</B>
  • <BR>: used to break the line in a html code. 
    <BR>Welcome to BathSpa. 
     Welcome to BathSpa.</BR>
  • <A>: Anchor tag is used to define hyperlinks. 
    <A href="https://bathspa.ac.ae/"</A>
  • <CENTER>: used to aling the text to center. 
    <CENTER>Welcome to BathSpa.</CENTER>
  • <ul>: used to define an unordered list. 
    <ul>
  <li>Menu A</li>
  <li>Menu B</li>
  <li>Menu C</li>
  <li>Menu D</li>
  </ul>
  • <HEAD>: used to define the heading of a document. 
    <HEAD>Welcome to BathSpa.</HEAD>

CSS styles

The below mentioned are some of the important css properties:

  • color: Used to set the color of a text.
  • text-align: Used to change the text allignment (center, left, right).
  • background-color: Used to change the background color.
  • margin: Helps to adjust the margin around a element.
  • border: Specifies the border of an element.
  • text decoration: Used to decorate the text.
  • font-size: Used to change the size of the font.
  • letter spacing: Used to increase or decrease the spacing between the text.
  • line height: Used to adjust hte height of the lines.
  • text transform: Used to control the letters.
